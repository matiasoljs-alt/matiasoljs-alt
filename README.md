## 📌 Featured Project

### 🔐 Basic User API

A simple API built with **FastAPI** and **SQLModel** that performs basic user management.

#### ✨ Features:
- Create users  
- List all users  
- Get a single user by ID  
- Delete users  

#### 📡 Endpoints:
- `POST /create_user` → Create a new user  
- `GET /ver_todos` → Get all users  
- `GET /pegar_um/{id}` → Get a specific user  
- `POST /deletar_user/{id}` → Delete a user  

#### 🛠 Tech used:
- Python  
- FastAPI  
- SQLModel  
- SQLite  

> ⚠️ Note: This is a simple project for learning purposes (no authentication/security implemented yet).
